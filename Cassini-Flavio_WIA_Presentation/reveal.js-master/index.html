Sliding Panels Template

Description

Sliding Panaels plugin is a simple portfolio template, with project preview images that slide out to reveal the selected project.

When coding this plugin I first looked at the JavaScript code in what I need to change to add more panels or take out a panel from the pre-configured 4 panels with in the code. After looking at the JavaScript I pursued to looking the HTML and CSS and found a better way to add more panels. Please check the next slide for the code.

To add another panel we have to first code the HTML. Go inside the index.html and on line 55 and add this code shown below. What your doing is adding the label for the for the panels labled project.

Code View

<li>
<a href="#0">
<div class="cd-project-title">
<h2>Project 5</h2>
<p>Brief description of the project here</p>
</div>
</a>
</li>

Now Still in the Index.html. Go to line 138 and add the code below. This will add the parapgraphs section when panel is clicked.

Code View

<li>
<div class="preview-image">
<div class="cd-project-title">
<h2>Project 5</h2>
<p>Brief description of the project here</p>
</div> 
</div>

<div class="cd-project-info">
<p>Text Goes Here....</p>
</div> <!-- .cd-project-info -->
</li>

Now go into the CSS folder and open style.css. In style.css you go to line 293 and change the width from 25% to 20%. This will make the panels shrink the width by 5% so you can add another panel. So if want to add 6 panels drop it by 15% and if you want to take one out add 5% making it 30%.

Code View

@media only screen and (min-width: 1024px) {
  .cd-projects-previews li {
    display: inline-block;
    height: 100%;
    width: 20%;
    float: left;
  }

Now still in the styles.css file go to line 392 and the code below. The code is where the 5th panel image will be added.

Code View

.cd-projects-previews li:nth-of-type(5) a,
.cd-projects > li:nth-of-type(5) .preview-image {
  background: #835531 url(../img/img-5.jpg) no-repeat center center;
  background-size: cover;

Now got to line 537 and 549 and add this code below.

Code View

.cd-projects > li:nth-of-type(5).selected .cd-project-title
